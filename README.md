# PgExamples
This is a TryIt style code previewer for SQL. It uses the [PGlite](https://pglite.dev/) WASM as the database engine, so you can run PostgreSQL in your browser. No server code needed.

Requires `node` and `npm`.

After downloading this and entering the directory, enter `npm install`. After it is successfully installed, you can run `npm run dev` to run it locally and `npm run build` to make a standalone distribution. 

There's a live demo at https://sean.brunnock.com/SQL/Examples/Basic/ .
