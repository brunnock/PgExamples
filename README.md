# PgExamples
This is TryIt style code previewer for SQL. It uses PostgreSQL as the database engine. All in the browser.

Requires `node` and `npm`.

After downloading this and entering the directory, enter `npm install`. After it is successfully installed, you can run `npm run dev` to run it locally.

There's a live demo at https://sean.brunnock.com/SQL/Examples/Basic/ .
